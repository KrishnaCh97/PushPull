My understanding of HTML

HTML stands for Hyper Text Markup Language. It describes the structure of a webpage.
It has three main elements
1.Root element
2.Head element
3.Body Element
Root element is the starting of the page, head element contains the header and footer elements of the page, whereas the body element contains the body of the page which is the main part of the HTML page.
The styles and formatting of the html pages can be using css or bootsrap.
